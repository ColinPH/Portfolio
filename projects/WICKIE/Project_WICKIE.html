The Game

This game is a 2D platformer featuring procedural level generation and an AI-driven enemy system. Built with Unity and C#, it leverages custom physics and shader effects for a dynamic experience.

Challenges

One of the major challenges was optimizing procedural generation to ensure both variety and playability. The solution involved using pathfinding algorithms to validate generated levels.

public void GenerateLevel() {
    for (int x = 0; x < width; x++) {
        for (int y = 0; y < height; y++) {
            if (Random.value > 0.7f) {
                PlaceTile(x, y);
            }
        }
    }
}

This function generates procedural levels while ensuring valid player paths.
